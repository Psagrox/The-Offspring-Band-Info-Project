# The Offspring Band Info Project

This repository contains my solution for the project about using the Spotify and YouTube APIs to bring information about my favorite band, "The Offspring". 

## Features
- Display information about The Offspring band.
- Fetch albums information using the Spotify API.
- Fetch last YouTube videos using the YouTube API.
- Display albums and videos in a responsive grid layout.
- Integrated with Tailwind CSS for styling.

## Technologies Used
- HTML
- CSS (with Tailwind CSS)
- JavaScript
- Spotify API
- YouTube API
